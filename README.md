
 # StudyFlow 📚

A personal productivity web app powered by Groq AI to help students manage and optimize their study sessions.

## 🌟 Features

- AI-powered study assistance via Groq API
- Clean, minimal interface for distraction-free studying
- Session planning and focus tools

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Groq API

## 🚀 Getting Started

1. Clone the repo
```bash
   git clone https://github.com/kanekiken263/studyflow.git
```

2. Open `index.html` in your browser

3. Add your Groq API key in the config section

## ⚠️ Important

Never hardcode your API key. Store it securely and add it via a config file that is listed in `.gitignore`.
